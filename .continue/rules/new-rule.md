# Project Architecture

This is a html5 js css application for brewery management

- Assets are in `/assets/`
- Data are in  `data`
- Lib are located in `/lib/`
- Javascript files are in `/scripts/`
- CSS are located in `/styles/`

## Coding Standards

- Follow the existing naming conventions
- Write tests for all new features
- Verify syntax
- Analyse impact of code changes
